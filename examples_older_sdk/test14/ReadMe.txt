yaneSDK2nd用サンプルその１４

αチャンネル付き可逆圧縮フォーマットyga形式のリアルタイムフェード、
ユニバーサルトランジションでレンズフレアを表現するものです。

画像データは、WAFFLE制作の『HAPPY ほたる荘』より。
年末発売なので、よかったら買ってください＾＾；

このアーカイブ自体の転載等は許可しますが、画像の二次利用はご遠慮ください。
（ただし、data/light.jpgは、自由に使って構いません）

ソースに関しては、自由に引用してご利用ください。
ソースのコンパイルには、yaneSDK2nd 1.54以降が必要です。
