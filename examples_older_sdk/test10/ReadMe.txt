セルオートマトンとは、ピクセル１つ１つが何らかの規則を持っていて、
その規則に基づいて次の状態に移り変わっていくものです。

たとえば、上にスクロールしながらぼかすならば、

	＊□＊
　　■＊■

□の次の瞬間のピクセル値(RGB)は、その左下と右下の２つ■のピクセル値(RGB)の
加重平均で決まるとします。

いろいろ応用は出来そうです。
面白い画面効果が出来たら、やねうらおまでメールで教えてください＾＾
